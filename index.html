Welcome to my personal web page!

I'm a cognitive and behavioral scientist with scientific training in advanced statistics, programming, and Psychology. I earned my Ph.D. degree after spending almost 5 years conducting research at William James Center for Research and Utrecht University. This work resulted in the publication of multiple research articles on scientific journals, which offered contributions to the fields of face and personality perception. After finishing my Ph.D., I kept collaborating in several research projects focused on face perception, for which I've been mainly contributing with data analysis and programming.

I built this website with some HTML and CSS editing of a free bootstrap template. Had lots of fun with this project, so I'm working on an improved version... coming soon!

    My Apps

    Random File Sampler

    • I wrote this program with the goal of automating the random sampling of face images from databases. This is useful to anyone looking to prevent any bias in image file selection. This program randomly samples (without replacement) files of any given extension from a folder. The sampled files are then copied to a new folder together with a .txt file listing the sampled file names. For reproducibility, the program uses a fixed pseudo-random generator seed number (seed=1234). Alternatively, you can set your own custom seed number. In the last updates, I added a user-friendly interface and created a Windows installer for the application.

    • For Windows users: click below to download the application installer
    • For Mac OS users: installer coming soon, as soon as I am able to compile on a Mac machine. If you are Python-savvy, download source code and run filesampler_app.py

    Face Masker: Real-time webcam face masking

    • This application applies a facial mask to any faces captured by the webcam. In addition, it includes a "Save snapshot" feature that captures the current frame and saves it as a JPEG file.
    • This was developed during my free time as a hobby project. There are some limitations in version 1.0 that I am still trying to overcome, especially the "laggy" frame rate. On top of that, I'm planning on packaging the app and distribute it with an installer.

    • Running the app: For now, the only way to run the Face Masker app is directly from the "[PROJECT_DIRECTORY]/src/facemasker_main.py" script.
